## 💻 Getting started

### Requirements

**Clone the project and access the folder**

```bash
$ git clone git@github.com:mauriciogirardi/play-the-old-game-react.git
```

**Follow the steps below**

```bash
# Install the dependencies
$ yarn

# Start the client
$ yarn start
```
